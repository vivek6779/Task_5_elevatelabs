# Task_5_elevatelabsDescription
->This project is a simple and accessible contact form built using HTML5 and CSS. It uses native browser validation without JavaScript to ensure correct user input.

# Features
->Uses semantic HTML elements (form, input, textarea, label, button)
->Proper input types (text, email, tel)
->HTML validation (required, pattern, maxlength)
->Accessible labels linked to inputs
->Clean and responsive UI with CSS
->Default browser validation messages

# Tools Used
->VS Code
->Browser DevTools
